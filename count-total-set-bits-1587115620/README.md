<h2>🧠 Intuition (the real idea)</h2>
<img src='https://img.shields.io/badge/Difficulty-Medium-yellow' alt='Difficulty: Medium' />
<hr>
<p><span style="font-size: 18px;">You are given a number<strong> n</strong>. Find the total count of set bits for all numbers from 1 to n (both inclusive).</span><br><br><span style="font-size: 18px;"><strong>Examples :<br></strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input</strong>: n = 4
<strong>Output</strong>: 5
<strong>Explanation</strong>: For numbers from 1 to 4. for 1: 0 0 1 =&gt; 1 set bit, for 2: 0 1 0 =&gt; 1 set bit, for 3: 0 1 1 =&gt; 2 set bits, for 4: 1 0 0 =&gt; 1 set bit. Therefore, the total set bits are 5.</span>
</pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: n = 17
<strong>Output</strong>: 35
<strong>Explanation</strong>: From numbers 1 to 17(both inclusive), the total number of set bits are 35.</span></pre>
<p><strong><span style="font-size: 18px;">Constraints:</span></strong><br><span style="font-size: 18px;">1 ≤ n ≤ 10<sup>8</sup></span></p>
